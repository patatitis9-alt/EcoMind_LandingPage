
# EcoMind_LandingPage

La landing page de ecomind presenta distintas vistas que explican el contenido dentro de la plataforma, intentando brindar el incentivo de participación a la ayuda ambiental, iniciando desde estudiantes y llegando a comunidades.

- Vistas disponibles:

    - Home
    - Preguntas frecuentes
    - Guía para padres
    - Comunidad
    - Unete ahora

## Segmentos objetivos

- Estudiantes de primaria (9-12 años)
- Padres de familia (25 - 50 años)
## Características

- Generales:
    - Titulos descriptivos
    - Navegación fluida
    - Tipografía Poppins
    - Diseño responsive
    - Relación: Colores - Ambiente

- Secciones:
    - Home: Presenta nuestro mensaje, breve resumen de funcionalidades y de quienes somos
    - Preguntas frecuentes: Responde las preguntas más comunes de nuestros usuarios en cuentos al uso de la plataforma
    - Guía para padres: Brinda consejos básicos de uso de la plataforma a los padres, para una mejor integración en su familia
    - Comunidad: Muestra distintas actividades disponibles para partipar junto a otros usuarios, como estadísticas relacionadas
    - Unete ahora: Sección de sing up, permite registro de cuentas, como también vincular con google o apple
